Recipes api

Get all recipes

  • URL
    • /recipes
  • METHOD
    • GET
  • URL PARAMS
    • None
  • QUERY PARAMS
    • None
  • Success Response
    • Code: 200
    • Content: { "1": { "category": "category name", "name": "recipe name", "preparation_time": "time in minutes" },...}
  • Error response
    • Code: 400
    • Content: {"code":400,"error":"URL is not a valid resource on this server"}

Get recipe by id

  • URL
    • /recipe/:id
  • METHOD
    • GET
  • URL PARAMS
    • Reguired: id=[integer]
  • QUERY PARAMS
    • None
  • Success Response
    • Code: 200
    • Content: { "category": "category name", "directions": [ "first step", "second step", ... ], "id": 1, "ingredient_list": [ { "amount": "amount in grams", "name": "ingrendient name" }, ... ], "name": "recipe name", "preparation_time": "time in minutes" }
  • Error response
    • Code: 400
    • Content: {"code":400,"error":"no recipe exist with id: ID"}
